¿Por qué utiliza el sitio la expresión “atracción hacia personas del mismo sexo”?

La “atracción hacia personas del mismo sexo” se refiere a la atracción emocional, física, romántica o sexual hacia una persona del mismo sexo. Las personas que experimentan atracción hacia personas del mismo sexo pueden optar o no por usar una designación de orientación sexual para describirse. De cualquier manera, “atracción hacia personas del mismo sexo” es un término técnico que describe la situación sin imponer una etiqueta. En este sitio web usamos este término para incluir a todas las personas que no se sienten cómodas usando una etiqueta, y no con la intención de negar la existencia de una identidad gay, lesbiana o bisexual.

Las etiquetas más comunes son lesbiana (una mujer que se siente atraída hacia las mujeres), gay (un hombre que se siente atraído hacia los hombres) y bisexual (un hombre o una mujer que sienten atracción hacia ambos sexos). Algunas mujeres utilizan también el término gay para describirse. Puede ser que las personas usen otras etiquetas, y algunos que experimentan atracción hacia personas del mismo sexo no utilizan ninguna etiqueta para describirse. La forma que las personas eligen para identificarse puede cambiar con el tiempo.
