Instrucciones para las asignaciones de soldadura

Instructor:

“El proyecto de soldadura debe entregarse el lunes. Busquen el metal que necesiten para el proyecto en el contenedor número 7. Suelden dos piezas en un ángulo de 45 grados. Quiero que pulan la soldadura hasta que quede prolija. Recuerden: cuanto mejor sea la soldadura, menos tendrán que pulir. Asegúrense de usar todo el equipo de protección; el otro día, vi que algunos de ustedes no lo tenían puesto. Les restaré puntos si no usan el equipo de protección completo”.
