Figura de las Escrituras

Para contar relatos de las Escrituras, esta figura puede representar a cualquier mujer de las Escrituras, como a Rebeca, a Sara o a Elisabet. Se puede colocar en cartulina gruesa, colorearla, recortarla, y después hacer con ella una marioneta con una varilla una figura para el franelógrafo o un títere hecho con una bolsa de papel, como se aprecia en la ilustración. Haz varias y a cada una coloréale el cabello y la ropa de manera diferente. El mes próximo aparecerá otra figura para las Escrituras.
