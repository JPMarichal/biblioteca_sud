Preparándonos para la Pascua de Resurrección

Jesucristo nos dio el ejemplo perfecto para que lo sigamos. Puedes usar esta actividad para aprender más acerca de Él y para prepararte para la Pascua de Resurrección. Empieza con el número 1, el domingo antes de la Pascua de Resurrección. Cada día, lee acerca de Jesús y contesta la pregunta; después, recorta la imagen que corresponde y añádela al cuadro.
