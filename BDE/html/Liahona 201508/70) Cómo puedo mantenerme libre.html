¿Cómo puedo mantenerme libre?

Dios quiere que seamos libres para elegir entre el bien y el mal.

Las adicciones a cosas como las drogas y el alcohol destruyen la buena salud y limitan nuestra libertad.

Pasar un tiempo excesivo con juegos de video, deportes o viendo la televisión pueden tomar todo tu tiempo libre;

es importante que conservemos algún tiempo libre para nuestra familia.
