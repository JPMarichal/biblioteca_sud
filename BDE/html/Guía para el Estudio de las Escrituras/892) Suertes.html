Suertes

Manera de hacer una elección o de eliminar varias posibilidades, lo cual a menudo se hace mediante la selección de un pedazo de papel o de un palito entre varios. A esto se le llama echar suertes.

  • Repartieron entre sí sus vestidos, echando suertes, Mateo 27:35 (Sal. 22:18; Mar. 15:24; Lucas 23:34; Juan 19:24).

  • Les echaron suertes, y la suerte cayó sobre Matías, Hech. 1:23–26.

  • Echamos suertes para ver cuál de nosotros iría a la casa de Labán, 1 Ne. 3:11.
